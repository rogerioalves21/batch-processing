batch-processing
================

API para processamento batch assíncrono baseado em jobs e steps.

O framework tem o objetivo simples de disparar processos assíncronos e com um mínimo de padronização. Baseado nos conceitos da [JSR 352](https://blogs.oracle.com/arungupta/entry/batch_applications_in_java_ee) e do [Spring Batch](http://spring.io/guides/gs/batch-processing/), a framework entra no conceito de Job e Step para processamentos batch.

Por favor leia nossa [Wiki](https://github.com/rogerioalves21/batch-processing/wiki) para mais informaçōes e exemplos.
